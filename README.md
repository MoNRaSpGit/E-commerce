# E-commerce Full Stack

Proyecto de e-commerce desarrollado con arquitectura full stack, enfocado en una base sólida, estructura clara y buenas prácticas.

## Tecnologías
- Frontend: React + Vite
- Backend: Node.js + Express
- Base de datos: SQL (a integrar)

## Descripción
El proyecto implementa un flujo básico de compra, pensado como base para futuras ampliaciones y mejoras.
